# vue-spinner

### usage：
```
<zk-spinner></zk-spinner>
```

### types：
* `<zk-spinner :type=0></zk-spinner>`：`snake`形式
* `<zk-spinner :type=1></zk-spinner>`：`doubl-bounce`形式

### parameters：
* `color`：默认为`red`
* `size`：默认为`1rem`

### example:
```
<zk-spinner :type=0 color="green" size="2"></zk-spinner>
```
将会生成一个以`snake`动画形式，颜色为`绿色`，宽度和高度为`2rem`的spinner
